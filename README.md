# [《副驾驶座上的盐粒》](https://weionxian.github.io/)

李珍的副驾驶座上有盐粒。

是海盐，粗粝，半透明，藏在座椅接缝里。每次急转弯，就能听见细微的沙沙声，像遥远海岸线的回响。

车子是银灰色SUV，中年人的标准配置。后座放着儿子的篮球和丈夫的高尔夫球包。只有副驾驶座是她的——严格来说，是十七年前那个她的。

盐粒来自东山岛的黄昏。2007年夏天，她和林海开一辆租来的吉普车环岛。海风灌满车厢，他的白衬衫鼓成帆。“开快点！”她赤脚踩在仪表台上，脚趾甲涂着贝壳粉色的漆。那个夏天的一切都沾着盐：亲吻的嘴唇，晒伤的肩胛，以及分手时没说出口的话。

后来她按部就班地生活，像所有人一样。结婚那天，婚车也是银灰色。丈夫体贴地帮她拉开车门，她忽然看见座椅缝隙有反光——一粒盐，不知怎么穿越一千公里跟来了。

“怎么了？”丈夫问。

“没什么。”她提起婚纱坐进去，小心避开那个位置。

盐粒很顽强。洗车、吸尘、暴晒，它总在某个雨天重新出现。像记忆本身，你以为擦掉了，其实只是融进了更深的地方。

去年冬天送儿子去补习班，等红灯时她忍不住摸索。找到了，在安全带插槽旁边。她捻起盐粒对着路灯看，灯光在其中折射出微型彩虹。后座的儿子突然说：“妈，你哭了？”

“没有，是雪。”

“可今天没下雪啊。”

是啊，没下雪。只是十七年前东山岛的某朵浪花，穿越时空落在了她的眼眶。

上个月同学会，听说林海移民了。照片上他抱着混血女儿站在枫树下，笑得很好。有人提起那个夏天，她低头搅动咖啡，勺子在杯沿碰出细响——和盐粒在座椅上滚动的声音一模一样。

回家的高速路上，丈夫难得开车。她坐在副驾驶座，看路灯一根根掠过。忽然想：如果当年跟去南方会怎样？但这个念头很快被碾碎，像车轮下的虚线。

车停进车库时，丈夫说：“这车该换了。”她下意识按住座椅：“再开两年吧。”

夜深人静，她又去车库。打开顶灯，用手机电筒照进每道缝隙。五粒、十粒、二十粒……她像考古学家发掘遗址。最大的一粒藏在扶手箱夹层，有米粒大，在灯光下像微型水晶。

她突然明白：这些不是同一粒盐。是海在持续不断地邮寄礼物，每个季节一粒，每个雨季一封，提醒她曾那么接近过飞翔。

凌晨三点，她把所有盐粒收集进香水瓶。摇晃时，声音像遥远的潮汐。瓶子放进首饰盒最底层，和婚戒放在一起。

第二天洗车，工人说：“李女士，您副驾驶座真干净。”

她笑了。阳光很好，仪表盘反射的光在挡风玻璃上画出彩虹。这一次，没有谁看见。
